## Cut-Han项目介绍
这是一个面向中文社区的视频剪辑工具，目前针对的用户主要是剪辑小白和中度使用者。

主要剪辑短视频为主，5到10分钟。

后续会逐渐增加AI功能，类似chatcut那样。



## 如何启动

启动网页服务器

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
